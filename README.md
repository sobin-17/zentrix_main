# Zentrix Technology Website

Official website for Zentrix Technology.

## Overview

This repository contains the source code for the Zentrix Technology corporate website, built using React and Vite.

The website serves as the digital presence of Zentrix Technology and showcases the company's services, solutions, innovations, and brand identity.

## Technology Stack

* React
* Vite
* JavaScript
* CSS
* Firebase Hosting
* Firebase Analytics
* Firestore

## Features

* Modern responsive design
* Mobile-first experience
* Interactive UI animations
* Performance-optimized frontend
* SEO-friendly architecture
* Firebase-powered deployment

## Development

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Create production build:

```bash
npm run build
```

Deploy to Firebase:

```bash
firebase deploy
```

## Deployment

The website is hosted using Firebase Hosting.
